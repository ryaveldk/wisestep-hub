# Manuskripter

Her samler vi tour-manuskripter i takt med at de skrives. Hver tour får sin egen markdown-fil under `/content/manuskripter/` og bliver bygget til en underside ved næste `npm run build`.

## Status

- **Kierkegaards København** — ikke påbegyndt. Venter på at manuskript-længde per stop og dramaturgisk struktur er afklaret (se [Fase 3](../faser/03-indhold.html)).

## Format (udkast)

Hvert manuskript følger denne struktur:

1. **Stop-nummer + lokation** (koordinater + landmark)
2. **Intro-kue** (hvornår geofence udløses, overgang fra forrige stop)
3. **Hovedspor** (dramatiseret fortælling, 3–5 min)
4. **Kilde-note** (hvilke værker/breve, til credits)
5. **Outro-kue** (rutehint til næste stop)

Manuskripter skrives på dansk først. Engelsk version kommer når vi har valgt en oversættelses-pipeline.
